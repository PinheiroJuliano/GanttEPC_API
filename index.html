Como hospedar para acesso permanente

Para um link fixo sem precisar regerar, hospede este arquivo HTML em qualquer lugar estático:

GitLab Pages — coloque o arquivo como public/index.html e ative Pages. O script Python pode rodar no CI para manter o CSV atualizado automaticamente:

pages:
  stage: deploy
  script:
    - pip install requests
    - python gera_csv.py
    - mkdir public
    - cp gantt_epc.html public/index.html
    - cp issues_EPC.csv public/
  artifacts:
    paths: [public]
  only: [main]

GitHub Pages / Netlify / Vercel — arraste o arquivo HTML. Gratuito e imediato.
